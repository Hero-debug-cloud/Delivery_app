## Radius Tokens

| Token | Value | Usage |
| :--- | :---: | :--- |
| `radius.sm` | 8px | Inputs, small controls |
| `radius.md` | 12px | Buttons, small cards |
| `radius.lg` | 24px | Cards, modals |
| `radius.xl` | 32px | Large visual panels |
| `radius.full` | 999px | Pills, avatars |
