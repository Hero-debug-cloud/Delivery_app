# Global Typography System

## Font Family

Use **Inter** across web and admin surfaces.

## Type Scale

| Token | Size | Line Height | Weight | Usage |
| :--- | :---: | :---: | :---: | :--- |
| `display.48.bold` | 48 | 56 | 700 | Desktop hero/auth marketing headlines |
| `heading.32.bold` | 32 | 40 | 700 | Desktop page/form titles |
| `heading.24.bold` | 24 | 32 | 700 | Mobile page/form titles |
| `heading.20.semibold` | 20 | 28 | 600 | Card titles |
| `body.16.regular` | 16 | 24 | 400 | Body text |
| `body.16.medium` | 16 | 24 | 500 | Labels, navigation, links |
| `caption.14.regular` | 14 | 20 | 400 | Helper text, metadata |
| `caption.14.medium` | 14 | 20 | 500 | Small buttons, badges |
| `tiny.12.medium` | 12 | 16 | 500 | Compact badges, table metadata |
