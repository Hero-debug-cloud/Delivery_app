# Payroll & Financial Settlement — Wireframe & Layout Specifications

This directory contains Figma-ready visual wireframes, interaction specifications, and component designs for the **Per-Store Payroll & Financial Settlement** module in LogiRoute.

## Module Structure

The payroll feature contains three core screens:
1. **[Payroll Console](file:///Users/me/Projects/delivery_app/specs/wireframes/payroll/payroll-console.md)** (`/payroll`) — The main overview dashboard and generation workspace.
2. **[Payout Ledger](file:///Users/me/Projects/delivery_app/specs/wireframes/payroll/payout-ledger.md)** (`/payroll/payouts`) — Weekly payout logs, detailed order breakdown drawer, hold policies, and bank CSV export tool.
3. **[Payroll Settings](file:///Users/me/Projects/delivery_app/specs/wireframes/payroll/payroll-settings.md)** (`/payroll/settings`) — Global default parameters and store-wise custom pricing override forms.
